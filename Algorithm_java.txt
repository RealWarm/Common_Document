toCharArray() >> 문자열을 문자배열로 만들어서 리턴한다
toUpperCase() >> 대문자로 전환

scanner.next() : 공백을 기준으로 토큰을 입력받음(띄어쓰기 \\s 기분)
scanner.nextLine() : 라인을 기준으로 문자열 입력 받음(줄넘김 \n 기준)
scanner.hasNext() : 다음 공백을 기준으로 문자열이 있는지 확인 후 boolean을 리턴
scanner.hasNextLine() : 다음 엔터를 기준으로 문자열이 있는지 확인 후 boolean을 리턴
